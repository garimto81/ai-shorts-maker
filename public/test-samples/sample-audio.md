# 테스트용 샘플 음성 파일

이 디렉토리에는 음성+비디오 합성 테스트를 위한 샘플 파일들이 있습니다.

## 사용 방법

1. **TTS 테스트**: 
   - 테스트 페이지에서 "TTS 테스트" 버튼을 클릭하여 AI 음성을 생성합니다.

2. **파일 업로드 테스트**:
   - 준비된 음성 파일을 업로드하여 테스트합니다.
   - 지원 형식: MP3, WAV, M4A, AAC, OGG

## 샘플 텍스트 (TTS 테스트용)

다음은 TTS 테스트에 사용할 수 있는 샘플 텍스트들입니다:

### 기본 테스트
"안녕하세요! AI 쇼츠 메이커 음성 테스트입니다. 깨끗하고 명확한 한국어 음성이 잘 생성되는지 확인해보세요."

### 감정 테스트
"와! 정말 놀라운 기술이네요! AI가 이렇게 자연스러운 음성을 만들 수 있다니 믿기지 않아요!"

### 설명형 테스트
"이 비디오에서는 AI 쇼츠 메이커의 음성 합성 기능을 테스트합니다. 텍스트를 입력하면 자동으로 자연스러운 한국어 음성이 생성됩니다."

## 테스트 체크리스트

- [ ] TTS 음성 생성 정상 작동
- [ ] 음성 파일 업로드 정상 작동
- [ ] 이미지와 음성 동기화 확인
- [ ] 다양한 음성 형식 지원 확인
- [ ] 에러 처리 정상 작동
- [ ] 전체 워크플로우 테스트 완료

## 문제 해결

### 음성이 생성되지 않는 경우
1. Azure Speech API 키 확인
2. 인터넷 연결 상태 확인
3. 텍스트 길이 확인 (너무 길거나 짧지 않은지)

### 파일 업로드가 실패하는 경우
1. 파일 형식 확인 (MP3, WAV 등)
2. 파일 크기 확인 (50MB 이하)
3. 브라우저 호환성 확인

### 비디오 렌더링이 실패하는 경우
1. FFmpeg 설치 상태 확인
2. 이미지 파일 존재 여부 확인
3. 시스템 리소스 확인
